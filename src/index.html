About ifShell

Click Add Game to find an Interactive Fiction game to play.

If you're new to Interactive Fiction, I recommend So, You've Never Played a Text Adventure Before, Huh? by Ryan Veeder to help get yourself acquainted. Some good, geeky historical games you might want to check out are Zork and The Hitchhiker's Guide to the Galaxy.

Click here to get started with a playlist of highly acclaimed contemporary games.

ifShell is a game management shell for Parchment and TADS. Game listings are obtained from the IFDB. The source code is available on Github, where the author may also be reached.
